* Acerca de Github Pages

/GitHub Pages/ es un servicio de alojamiento de páginas web estáticas que recopila archivos HTML, CSS y JavaScript directamente desde un repositorio albergado en GitHub, existiendo la posibilidad de que el servicio ejecute de forma opcional estos archivos a través de un proceso de construcción, para posteriormente publicarlos como un sitio web.

#+BEGIN_QUOTE
*Nota:* Por defecto, Github Pages aloja la totalidad de sus sitios web bajo el nombre de dominio =github.io=; sin embargo, es posible reemplazarlo por uno propio si así se desea.
#+END_QUOTE
